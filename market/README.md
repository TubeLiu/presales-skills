# market plugin — B2B 科技营销文案编辑

对现有文案进行系统性七轮逐维度编辑，提升清晰度、说服力和转化率。

适用于产品介绍、方案封面、客户案例、官网落地页、招投标商务文件、邮件模板等 B2B 科技营销场景。

## Slash 入口

| 触发方式 | 形式 |
|---|---|
| Claude Code canonical | `/market:polish` |
| Codex / Cursor / OpenCode 短形式 alias | `/polish` |
| 自然语言 auto-trigger | "审一下文案" / "编辑营销文案" / "copy edit" / "polish this" / "润色一下" / "太冗长了" / "精简文案" / "内容过时了" |

## 适用场景

- 产品介绍页的文案优化
- 方案封面摘要的精简和增强
- 客户案例的叙事打磨
- 官网落地页的转化优化
- 招投标商务文件的措辞润色
- 售前邮件和客户跟进邮件的编辑
- 过期内容的刷新和更新

## 七轮编辑框架

| 轮次 | 聚焦维度 | 核心问题 |
|------|---------|---------|
| 第一轮 | 清晰度 | 读者能理解你在说什么吗？ |
| 第二轮 | 语气一致性 | 语调前后一致吗？ |
| 第三轮 | 价值关联 | 每个表述都回答了"跟我有什么关系"吗？ |
| 第四轮 | 证据支撑 | 每个论断都有证据支持吗？ |
| 第五轮 | 具体化 | 文案足够具体以令人信服吗？ |
| 第六轮 | 情感共鸣 | 文案让读者有感觉吗？ |
| 第七轮 | 风险消除 | 移除了所有行动障碍吗？ |

## 附带功能

- **专家团评分**——多视角打分 + 具体批评，适合高风险文案
- **快速编辑检查**——不需要完整七轮时的快速审查
- **中文冗余表达替代表**——B2B 科技文案常见八股 → 简洁替代
- **内容刷新框架**——过期内容的刷新 vs. 重写决策矩阵
