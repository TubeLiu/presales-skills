# 内容刷新框架

文案编辑不只是针对新内容。现有内容会随时间衰退——数据过时、案例陈旧、品牌语调漂移、错失 SEO 机会。内容刷新将同样的编辑严谨性应用于已发布的内容。

## 何时刷新

- **流量下降**——某个页面曾经表现好，现在不行了
- **数据过时**——统计数据超过 12 个月
- **产品已变更**——功能、定价或定位与当前不符
- **竞品更新**——竞品的内容已经超过你
- **品牌语调漂移**——早期内容的风格与现在不一致

## 刷新 vs. 重写决策矩阵

| 维度 | 刷新 | 重写 |
|------|------|------|
| 核心信息 | 仍然有效 | 需要改变 |
| 页面结构 | 逻辑通顺 | 需要重组 |
| 数据 / 案例 | 过时但可更新 | 已经不适用 |
| 品牌语调 | 略有偏差 | 完全不对 |
| SEO 表现 | 流量下降但有基础 | 从未有过流量 |
| 工作量 | 小（几小时） | 大（需要重新创作） |

**结论：** 6 项中 4 项以上指向"刷新"→ 刷新；3 项以上指向"重写"→ 重写。

## 内容刷新检查清单

### 1. 数据更新
- [ ] 所有统计数据不超过 12 个月
- [ ] 客户案例仍在合作或最近完成
- [ ] 产品截图是最新版本
- [ ] 价格和套餐信息准确
- [ ] 外部链接都有效

### 2. 信息准确性
- [ ] 产品功能描述与当前版本一致
- [ ] 技术术语使用准确（不过时）
- [ ] 竞品对比信息是最新的
- [ ] 合规和认证信息是最新的

### 3. 品牌一致性
- [ ] 语调与当前品牌指南一致
- [ ] 用词风格与近期内容匹配
- [ ] 视觉元素（logo、配色）是当前版本
- [ ] CTA 与当前营销漏斗一致

### 4. SEO 优化
- [ ] 标题标签仍针对正确的关键词
- [ ] Meta description 引人点击
- [ ] 内部链接指向当前活跃页面
- [ ] 结构化数据标记正确

### 5. 转化优化
- [ ] CTA 位置和文案仍然有效
- [ ] 表单字段数量合理
- [ ] 信任信号（客户 logo、证言）是最新的
- [ ] 移动端体验良好

## 内容更新节奏建议

| 内容类型 | 建议审查频率 |
|---------|------------|
| 产品介绍页 | 每次产品发布后 |
| 定价页 | 每次定价变更后 |
| 客户案例 | 每季度 |
| 博客文章 | 每 6 个月 |
| 技术文档 | 每次版本发布后 |
| 关于我们 | 每半年 |
| 落地页 | 每月检查转化数据 |

## 快速刷新流程

1. **通读**——像新读者一样读一遍，标记任何"感觉不对"的地方
2. **数据审计**——列出所有数据点，逐一验证
3. **竞品扫描**——看竞品同类型页面，找差距
4. **按需编辑**——用七轮编辑框架处理需要改的部分
5. **发布前检查**——走一遍上面的检查清单
